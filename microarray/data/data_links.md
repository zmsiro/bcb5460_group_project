# Microarray Data Source

Data were obtained from the Gene Expression Omnibus (GEO).

## Dataset information:
- Accession: GSE26024
- Organism: Mus musculus (mouse)
- Tissue: Striatum (mouse)

## Link for microarray data
https://www.ncbi.nlm.nih.gov/geo/query/acc.cgi?acc=GSE26024

## Notes
- Two platforms available:
  - Affymetrix (GPL1261)
  - Illumina (GPL6885)

- This analysis utilizes the Illumina dataset
